Camunda LLM Demo

Camunda with LLM Integration

Simplified Ticket Refund Business Process Demo

Portfolio Project by Igor Kudinov

Production-ready BPMN 2.0 workflow for automated airline ticket refund processing. Built with Camunda Platform 7 and powered by OpenAI GPT-4o-mini, showcasing intelligent fare rule analysis through the External Task pattern.

Key Features

  • Automated ticket refund workflow with BPMN 2.0
  • LLM-powered fare rule analysis using External Task pattern
  • Retry logic with error handling and boundary events
  • PostgreSQL database for process persistence
  • Custom Go-based history viewer for process monitoring
